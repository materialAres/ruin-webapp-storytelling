# Ruin

A dark, atmospheric, and introspective interactive story experiment.

## How it's made
This interactive site is just plain HTML, CSS and JavaScript.

## Features

* A dark and atmospheric story that explores the human psyche.
* A haunting soundtrack.
* A focus on introspection.
